Register: POST http://localhost:3000/auth/register
Body (JSON): {"username": "JohnDoe", "email": "john@example.com", "password": "password123"}
Login: POST http://localhost:3000/auth/login
Body (JSON): {"email": "john@example.com", "password": "password123"}
Logout: POST http://localhost:3000/auth/logout
📝 Notes APIs
Create Note: POST http://localhost:3000/api/notes
Body (JSON):
json
{
  "topic": "Photosynthesis",
  "subject": "Biology",
  "content": "Plants make food using sunlight...",
  "contentType": "text"
}
Get All Notes: GET http://localhost:3000/api/notes
Search Notes: GET http://localhost:3000/api/notes/search?q=Biology
Get Single Note: GET http://localhost:3000/api/notes/REPLACE_WITH_ID
Update Note: PUT http://localhost:3000/api/notes/REPLACE_WITH_ID
Delete Note: DELETE http://localhost:3000/api/notes/REPLACE_WITH_ID
📂 Resources APIs
Upload File: POST http://localhost:3000/api/resources/upload
Body: Use form-data, key file, and upload any file.
Share Resource: POST http://localhost:3000/api/resources/share
Body (JSON): {"resourceId": "...", "sharedWithUserId": "..."}
Tip: After you login once in Postman, you can test all the other routes. If you get an "Unauthorized" error, just run the Login API again.
