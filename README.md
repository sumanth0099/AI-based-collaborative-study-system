# 📚 AI-Based Collaborative Study System

An AI-powered collaborative learning platform built to help students organize notes, share study resources, collaborate with friends, and use AI-powered study tools—all in one place.

## 🚀 Live Demo

**Website:** https://ai-based-collaborative-study-system-1u1c.onrender.com/

---

## ✨ Features

### 🏠 Home

* View notes shared by users
* AI-generated learning insights

### 📊 Dashboard

* Personal notes overview
* Quiz statistics
* Strong topics analysis
* Shared resources summary

### 📝 Notes

* Create and manage text-based notes
* Use notes as input for AI tools

### 📂 Resources

* Upload study materials to the cloud
* Share resources with friends

### 🤖 AI Tools

Generate study material directly from your notes:

* AI Quiz Generator
* AI Flashcards
* AI Chat
* AI Summary
* Key Question Generator

### 📰 Education News

Latest educational news powered by Google News RSS across topics including:

* Scholarships
* Admissions
* Exams
* Jobs & Internships
* Government Updates
* Financial Aid
* Study Abroad
* Free Courses
* Competitions
* Research

### 👥 Groups

* Create public and private study groups
* Join public groups
* Request access to private groups
* Real-time group chat
* Member management

### 🤝 Friends

* Search for users
* Send and receive friend requests
* Manage friends
* One-to-one real-time chat

### 🔔 Notifications

Receive notifications for:

* Friend requests
* Group join requests
* Messages
* Other important activities

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

```bash
cd ai-based-collaborative-study-system
```

### Install dependencies

Backend

```bash
cd backend
npm install
```

Frontend

```bash
cd frontend
npm install
```

### Configure Environment Variables

Create `.env` files in both the backend and frontend folders.

```env
PORT=

DATABASE_URL=

SESSION_SECRET=

GOOGLE_CLIENT_ID=

GOOGLE_CLIENT_SECRET=

REDIS_URL=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=

AI_API_KEY=
```

### Start the project

Backend

```bash
cd backend
node server.js
```

Frontend

cd frontend
npm run dev
